INSTRUCTIONS TO USE:
simply run the index.html in a live server and play the game!

The arrow keys control the movement.
The key 'c'  will switch between the maps.
The keys '(' and ')' will rotate the pacman at it's own location.
The keys '{' and '}' will rotate the whole grid.
The key 'm' will allow the user to go into modify mode, teleport the pacman to new cordi and then exit the modify mode on pressing 'm' again.
